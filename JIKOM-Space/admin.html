Cara Pakai Admin Panel

Cara Upload Gambar

  1. Buka drive.google.com
  2. Upload gambar dari perangkatmu
  3. Klik kanan gambar → pilih "Bagikan"
  4. Ubah akses ke "Siapa saja yang memiliki link" → klik Selesai
  5. Salin link dari Google Drive
  6. Buka menu Tools > Link GDrive di sidebar kiri
  7. Paste link Google Drive tadi, lalu klik "Generate Link"
  8. Copy "Link Gambar Langsung" yang muncul dan gunakan di aplikasi

Cara Atur Gambar dalam Konten

  1. Di editor konten, klik tombol "+ Sisipkan Gambar" di toolbar
  2. Paste URL gambar, lalu pilih posisi: Kiri, Kanan, Tengah, atau Penuh
  3. Atur ukuran gambar dengan slider
  4. Klik "Sisipkan" maka gambarnya akan langsung muncul di editor
  5. Klik gambar di editor untuk mengedit ulang posisi atau ukuran gambar

Cara Publish ke Website

  1. Setelah selesai tambah atau edit konten, klik menu Generate data.js
  2. Klik tombol "Salin Semua"
  3. Buka github.com → masuk ke repo jikomspace
  4. Buka file data.js → klik ikon pensil (Edit)
  5. Hapus semua isi → paste kode yang sudah disalin
  6. Scroll bawah → klik "Commit changes"
  7. Tunggu ~30 detik → website langsung update otomatis!

Penting Diingat

  • Gambar yang kamu sisipkan harus dari URL publik (Google Drive share link). Jangan pakai file lokal.
  • Format tanggal: DD Month YYYY contoh: 16 Juni 2026
